Chapter 2. Создание новых тем для Same GNOME

Сделать новую анимированную тему для игры Same GNOME очень просто. Каждый элемент должен быть размером 40 на 40 пикселов. Вам надо предоставить изображение, которое содержит хотя бы 3 строки по 40 пикселов каждая (для элементов трех различных цветов) и произвольное число кадров создающих вашу анимацию (эта анимация будет показываться когда мышиный курсор будет находиться над набором шариков того же цвета).

Чтобы дать возможность игре использовать ваш набор картинок, вам надо установить его в каталог $prefix/pixmaps/same-gnome (где $prefix означает установочный префикс используемый для конфигурирования окружения GNOME, вы можете узнать его выполнив команду gnome-config --prefix).
