Chapter 2. Making new themes for Same GNOME

Making new animation themes for Same GNOME is very simple. Each tile should measure forty by forty pixels. You need to provide a graphic image that contains at least 3 rows of 40 pixels each (for the three different tile colors) and a variable number of frames to do your animation (this animation will be shown when the mouse cursor is over a cluster of tiles of the same color).

To allow Same-GNOME to pick up your tile set you need to install this on the $prefix/pixmaps/same-gnome directory (where $prefix stands for the installation prefix used to configure the GNOME environment, you can find this out by running the command gnome-config --prefix).
